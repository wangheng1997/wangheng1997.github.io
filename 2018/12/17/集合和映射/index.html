集合和映射

发表于 2018-12-17 | 阅读次数:
字数统计: 954 | 阅读时长 ≈ 4

Set集合

集合(set)是一个用于存储和处理无重复元素的高效数据结构。

  1. 集合

  2. 分为有序集合和无序集合。

    –有序结合基于搜索树实现。

    –无序集合基于哈希表实现。

  3. 不能添加重复元素。
  4. 多重集合可以添加重复元素。

接口如下：

  1. void add(E e); //添加元素E
  2. void remove(E e); //删除元素E
  3. boolean contains(E e); //是否包含元素E
  4. int getSize(); //返回集合长度
  5. boolean isEmpty(); //判断是否为空

Map映射

映射表(map)类似于目录，提供了使用键值快速查询和获取值的功能。
可以使用链表和二叉树来实现。

有序映射 ： 搜索树实现。

无序映射： 哈希表实现。

多重映射： 键可以重复。

无序映射
定义接口:

  1. void add(K,v); // 添加元素
  2. V remove(K); //删除元素
  3. boolean contains(K); //是否包含元素
  4. V get(K); //得到元素
  5. void set(k,v); // 修改元素
  6. int getSize(); //元素个数
  7. boolean isEmpty(); //判断是否为空

总结：

  1. 集合存储的是不重复的元素，映射表中存储的是键/值对。

  2. Map将键映射到元素上，键类似于索引，Map接口提供了查询、更新以及获取值的集合以及键的集合的方法。

  3. Java合集框架支持三种类型的集合：HashSet、LinkedHashSet、TreeSet。

  4. Java合集框架支持三种类型的映射表：HashMap、LinkedHashMap、TreeMap。
