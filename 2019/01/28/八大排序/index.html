1. 插入排序

基本思想： 将一个记录插入到一个有序表中，从而得到一个新的有序表。即将第一个记录当成一个有序表，然后进行插入。

如果碰到相等的元素，则把插入元素放到被插入元素的后面。相等元素的顺序没有改变，所以插入排序是稳定的。

2. 希尔排序

基本思想： 先将整个待排序的记录分割成为若干个子序列分别进行排序，待整个序列记录基本有序时，再对全体记录依次直接插入排序。

操作方法：

  1. 选择一个增量序列t1、t2、….tk。其中t(k)=1，且t(i)>t(i-1)
  2. 按增量序列个数K，对序列进行K趟排序。
  3. 每趟排序，根据增量t(i),将待排序列分割成若干长度为m的子序列。分别对各子表进行插入排序。
  4. 增量为子序列中数据与数据的间距。

基本思想： 在一组数据中，选出最小值(最大值)和第1个交换，然后在剩下的数据中继续找出最小值，与第2个交换，以此类推，直到n-1和第n个值交换。

4. 堆排序

基本思想：

先介绍一下堆，堆是一颗完全二叉树，有最大堆和最小堆，以最大堆为例，父节点的值大于左右孩子结点。

首先，要把排序时的n个数据建为一个堆， 然后将堆首元素和最后一个元素互换。此时，对前n-1个数据重新调整建堆，再将堆首元素和最后一个元素互换(此时堆为前n-1个元素，即最后一个元素为n-1)，以此类推，最后得到n个有序的序列。

基本操作：

  1. 如何将n个数据建成堆。(参考siftDowm函数)
  2. 交换元素后，如何给n-1个元素重新排序。(参考siftDown函数的length参数，控制排序数组的长度即可)

分析：

设树深度为K，则比较次数不超过2(k-1)次，交换次数至多K次，所以排序的复杂度为0(NlogN)

5. 冒泡排序

基本思想：

对一组未排序的数据，自上而下对相邻的两个数进行比较，较小的数据往上冒，较大的数据往下沉。

6. 快速排序

基本思想：

  1. 选择一个基准元素，通常选择第一个元素，或随机选取数据中一个元素。
  2. 通过一趟排序将待排序的记录分割成两部分，其中一部分记录比基准元素小，另一部分比基准元素打。
  3. 此时基准元素在排序好的正确位置上。
  4. 然后对这两部份进行相同的排序操作。

改进算法： 双路快速排序算法

由于数据本身可能有序或大部分有序且重复，导致基准元素总是在最前面和最后面。最坏的情况复杂度为O(n^2)

具体步骤：

  1. 使用两个索引值从左边和右边开始遍历。
  2. 将小于基准元素的值放在左边，大于基准元素的值放在右边。
  3. 将重复的值均匀分布到两端，这样尽可能使基准元素处在中间位置。

改进算法：三路划分快速排序算法
当待排序列中有大量重复数据时，可以使用三路划分，左边为小于基准元素，右边为大于基准元素，中间为等于基准元素。大大减少的重复元素排序的时间。

7. 归并排序

基本思想：

归并（Merge）排序法是将两个（或两个以上）有序表合并成一个新的有序表，即把待排序序列分为若干个子序列，每个子序列是有序的。然后再把有序子序列合并为整体有序序列。

8. 桶排序/基数排序

基本思想：

按照低位先排序，然后收集，然后再依次向高位排序，再收集，直到最高位。基数排序基于分别排序，分别收集，所以是稳定的。

插入 O(n2) O(n2) 稳定 O(1) 大部分已排序时较好
基数 O(logRB) O(logRB) 稳定 O(n) B是真数(0-9),R是基数(个十百)
