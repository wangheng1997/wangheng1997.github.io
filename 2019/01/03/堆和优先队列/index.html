优先队列

出队顺序和入队顺序无关，和优先级有关。使用堆这种数据结构来实现优先队列(出队和入队时间复杂度是logn)。

堆

二叉堆

最小堆：堆中的每个结点总是小于父结点的值。
最大堆：堆中的每个结点总是不大于父结点的值。

接口：

  1. int getSize(); //得到元素个数
  2. boolean isEmpty(); //判断是否为空
  3. add(E e);//向堆中添加元素
  4. remove(E e); //从堆中删除元素
  5. get(int index); //从堆中查询元素

replace操作

heapify操作

将一个数组变成堆的结构。从二叉堆中最后一个非叶子节点开始，向前遍历，进行下沉。
